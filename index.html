SAP SSO 자동 로그인 가이드

쉽고 빠르게, 세아웍스 포털을 통해 SAP 시스템에 로그인하세요.
복잡한 비밀번호 입력 없이, 클릭 한 번으로 업무를 시작할 수 있습니다.

- 디지털 전략팀 -

1. 세아웍스 퀵링크

SeAH Works 포털에 접속 후, 바로가기 메뉴의 '퀵링크'를 선택합니다.

  • (이미지 1번) 퀵링크 메뉴 목록에서 'SAP 로그인' 항목을 클릭하여 접속합니다.

2. SAP 자동 로그인 최초 설정

처음 자동 로그인을 설정하는 PC에서 **딱 한 번**만 진행하는 단계입니다.

  1. (이미지 1번) 화면 중앙의 파란색 'SAP 프로그램... REG 파일 다운로드' 버튼을 클릭합니다.
  2. (이미지 2번) 다운로드된 `SapShcutLoginReg.bat` 파일을 클릭하여 실행합니다.
  3. (이미지 3번) Windows 보안 경고창에서 '실행' 버튼을 클릭합니다.
  4. (실행 확인) 검은색 명령(cmd) 창이 자동으로 사라지면 설정이 완료됩니다.

3. SAP 자동 로그인 프로세스 (설정 이후)

최초 설정을 마쳤다면, 앞으로 SAP에 접속할 때는 이 방법만 사용하시면 됩니다.

  1. (이미지 1번) '항상 ... 열기' 체크박스를 선택하면 다음 접속부터 편리합니다.
  2. (이미지 2번) 'Windows 명령 처리기 열기' 버튼을 클릭합니다.
  3. (이미지 3번) 검은색 명령(cmd) 창이 나타났다가 자동으로 사라집니다.
  4. (이미지 4번) SAP 시스템에 자동으로 로그인됩니다.
