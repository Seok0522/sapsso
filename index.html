SAP SSO 자동 로그인 가이드

클릭 한 번으로 간편하게 SAP 시스템에 접속하세요.
최초 설정 이후에는 비밀번호 입력 없이 바로 업무를 시작할 수 있습니다.

  1. 1 퀵링크 접속
  2. 2 최초 설정
  3. 3 이후 사용법
단계 1

세아웍스 퀵링크 접속

SeAH Works 포털에 접속하여 '바로가기' 메뉴의 '퀵링크'를 선택한 후, 'SAP 로그인' 항목을 클릭하여 접속합니다.

'SAP 로그인' 퀵링크를 클릭하세요.

이미지를 참고하여 정확한 위치를 확인하세요.

단계 2

SAP 자동 로그인 최초 설정 (1회만)

처음 자동 로그인을 설정하는 PC에서 **딱 한 번**만 진행하는 단계입니다. 아래 순서대로 따라해 주세요.

  1. REG 파일 다운로드

    화면 중앙의 파란색 'SAP 프로그램... REG 파일 다운로드' 버튼을 클릭합니다.

  2. 파일 실행

    다운로드된 `SapShcutLoginReg.bat` 파일을 클릭하여 실행합니다.

  3. 보안 경고창 '실행' 클릭

    Windows 보안 경고창에서 '실행' 버튼을 클릭하면 설정이 완료됩니다. ✅

단계 3

간편한 자동 로그인 (설정 이후)

최초 설정을 마쳤다면, 앞으로는 아래의 간단한 과정만으로 SAP에 바로 접속할 수 있습니다.

  1. '항상 열기' 선택 (권장)

    '항상 ... 열기' 체크박스를 선택하면 다음 접속부터 더욱 편리합니다.

  2. 명령 처리기 열기

    'Windows 명령 처리기 열기' 버튼을 클릭합니다.

  3. 자동 로그인 완료

    잠시 후 SAP 시스템에 자동으로 로그인됩니다. 🎉
