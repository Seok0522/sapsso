SAP SSO 자동 로그인 가이드

쉽고 빠르게, 세아웍스 포털을 통해 SAP 시스템에 로그인하세요.
복잡한 비밀번호 입력 없이, 클릭 한 번으로 업무를 시작할 수 있습니다.

- 디지털 전략팀 -

단계 1

세아웍스 퀵링크 접속

SeAH Works 포털에 접속하여 '바로가기' 메뉴의 '퀵링크'를 선택한 후, 'SAP 로그인' 항목을 클릭하여 접속합니다.

단계 2

SAP 자동 로그인 최초 설정 (1회만)

처음 자동 로그인을 설정하는 PC에서 **딱 한 번**만 진행하는 단계입니다. 아래 순서대로 따라해 주세요.

  1. REG 파일 다운로드

    화면 중앙의 파란색 'SAP 프로그램... REG 파일 다운로드' 버튼을 클릭합니다.

  2. 파일 실행

    다운로드된 `SapShcutLoginReg.bat` 파일을 클릭하여 실행합니다.

  3. 보안 경고창 '실행' 클릭

    Windows 보안 경고창에서 '실행' 버튼을 클릭하면 설정이 완료됩니다. ✅

단계 3

간편한 자동 로그인 (설정 이후)

최초 설정을 마쳤다면, 앞으로는 아래의 간단한 과정만으로 SAP에 바로 접속할 수 있습니다.

  1. 퀵링크 클릭

    접속을 원하시는 세아웍스 퀵링크를 클릭하세요.

  2. '항상 열기' 선택 (권장)

    '항상 ... 열기' 체크박스를 체크하고 windows 명령 처리기 열기 버튼을 눌러 주세요.
    → 체크 해 주시면, 다음 접속부터 퀵링크 클릭 시 이 팝업 창이 뜨지 않고 바로 SAP 로그인이 완료 됩니다.

  3. 자동 로그인 완료

    잠시 후 명령 처리기가 자동으로 실행되며 SAP 시스템에 자동으로 로그인됩니다. 🎉

Q&A

자주 묻는 질문

Q. SSO 적용 후 SAP 로그인이 되지 않습니다. 어떻게 해야 하나요?

- SSO는 SAP 최신 버전을 기준으로 작동하기에, 일부 구 버전 사용자 분들은 접속 시 이슈가 발생할 수 있습니다.

- 만약 로그인이 되지 않으신다면, VNTG 제조ERP2팀 홍영채 매니저에게 문의 부탁 드립니다.

Q. SSO 자동 로그인 기능 사용 시, 기존 접속 방식인 ID,PW 입력 방식의 로그인 사용이 가능할까요?

- 불가능합니다. 기존에 업연을 통해 연락드린 내용이나, 세아웍스 사용 불가 또는 공용 계정 이슈로 기존 ID/PW 접속 방식으로의 변경을 원하신다면 아래 연락 창구로 연락 부탁 드립니다.

- 세아베스틸 : 디지털전략팀 양서윤 대리

- 세아창원특수강 : 디지털전략팀 조민경 대리

Q. SSO 관련 문의가 있습니다. 어떤 분에게 연락하면 될까요?

- 기능 관련 문의는, VNTG 제조 ERP 2팀 홍영채 매니저 또는 디지털 전략팀 손석 대리에게 문의 바랍니다.

- SSO 대상 계정 변경 문의는 (세아웍스 사용 불가 또는 공용 계정으로 SSO 미 적용 필요) 세아베스틸 디지털전략팀 양서윤 대리, 세아창원특수강 디지털전략팀 조민경 대리에게 문의 바랍니다.

Q. SSO 적용 대상 계정은 전 사 임직원인가요?

- SSO 자동 로그인 방식은 현재 SAP 사용자 분들을 대상으로 시행 됩니다. (현재 SAP 미 사용자는 대상에서 제외) 이 점 양해 부탁드립니다.
